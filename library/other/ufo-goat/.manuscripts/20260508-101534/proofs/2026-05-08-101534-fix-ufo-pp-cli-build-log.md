# UFO CLI Build Log

## What was built

### Priority 0 (Foundation)
- Custom CSV manifest parser (`internal/manifest/manifest.go`)
- Custom SQLite data layer (`internal/store/ufo_store.go`) with UpsertUFOFileBatch, ListUFOFiles, SearchUFOFiles, GetTimeline, GetPairs, GetLocations, GetAgencySummary, GetNewFiles, MarkDownloaded, RebuildFTS, EnsureUFOSchema
- FTS5 full-text search index on titles, descriptions, locations

### Priority 1 (Absorbed features)
- `sync` — Fetches CSV from GitHub, parses, upserts 162 files into SQLite
- `files list` — Filters: --agency, --type, --location, --after, --before, --redacted, --limit
- `files get` — Lookup by ID or partial title match
- `files search` / `search` — FTS5 search across titles, descriptions, locations
- `agencies` — Breakdown with PDF/VID/IMG counts and date ranges
- `export` — JSONL/JSON export (generated)
- `doctor` — Health check (generated + customized)

### Priority 2 (Transcendence features)
- `timeline` — Chronological incident timeline, sorted by parsed date
- `pairs` — Video-PDF pairings (21 pairs found)
- `new` — Files added since last sync, with --since support
- `download` — Download manager with --resume, --agency, --type filters, 403 handling
- `locations` — Location aggregation with counts, agencies, date ranges

### Priority 3 (Polish)
- Fixed CSV column name mismatch for "PDF | Image Link" (spaces around pipe)
- All commands support --json, --agent, --select, --compact, --csv flags

## Test Results
- Build: PASS (go build, go vet, go test all pass)
- Sync: 162 files fetched and stored (159 unique after dedup)
- Files list: 159 files, all filters work
- Search "Apollo": 11 results via FTS5
- Search "Iraq": 9 results
- Timeline: 98 dated incidents
- Pairs: 21 video-PDF pairings
- Agencies: 5 agencies with type breakdowns
- Locations: 36+ unique locations
- Videos: 28 VID-type files
- Download URLs: 143/159 populated

## Intentionally Deferred
- `--limit` flag on timeline, pairs, locations (not generated by framework)
- DVIDS API integration for alternate video downloads
- Browser cookie import for war.gov downloads (auth login --chrome)
