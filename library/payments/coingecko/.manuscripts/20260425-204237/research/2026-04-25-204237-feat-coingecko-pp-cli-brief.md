# CoinGecko CLI — research brief

- Spec source: community
- Spec URL: https://docs.coingecko.com/v3.0.1/reference/introduction
- CLI: `coingecko-pp-cli`
- MCP: `coingecko-pp-mcp`

CoinGecko does not publish a public OpenAPI spec; this generated CLI uses a hand-written OpenAPI 3.0.3 spec covering 11 key free-tier endpoints from public docs.
