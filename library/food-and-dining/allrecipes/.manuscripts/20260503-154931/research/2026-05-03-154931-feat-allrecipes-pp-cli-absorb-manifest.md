## Transcendence (only possible with our approach)

| 1 | Pantry match | `pantry --pantry-file <file> --query <q> --max-missing N` | 10/10 | SQL join over local recipes×ingredients with token-overlap scoring; ranks by `(have_count - missing_count)` and filters by `--max-missing` | Casey, Agent |
| 2 | Bayesian top-rated | `top-rated <query-or-category> --smooth-c 200 --limit N` | 10/10 | Pure SQL over cache: `(C*4.0 + sum_stars) / (C + reviewCount)`; orders desc | All four |

## Reachability Note (2026-05-03)

Cloudflare has tightened on recipe detail pages. Browse/search work with
plain stdlib HTTP; recipe details need a Cloudflare clearance cookie via
`auth login --chrome`. The `doctor` command (transcendence row #7) names
this failure mode by inspecting the `cf-mitigated: challenge` header.
